## Descripción
Este proyecto consiste en el desarrollo de una calculadora como trabajo práctico de la materia Ingeniería de Software I.

## Alcance inicial
En una primera etapa, la calculadora incluirá las siguientes operaciones:

- suma
- resta
- multiplicación
- división

Cada operación incluye su validación correspondiente.

## Objetivo
Desarrollar una solución funcional aplicando un proceso de trabajo organizado, contemplando relevamiento de requerimientos, diseño, implementación, pruebas y presentación final.

## Organización del trabajo

- Victoria Pereyra: multiplicación y división

## Estructura del proyecto
El proyecto está organizado en tres clases principales:

- `Main`: punto de entrada del programa
- `Menu`: muestra las opciones y controla la navegación
- `Logica`: contiene las operaciones y sus validaciones
